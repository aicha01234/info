Exercices

EXERCICE 1 :
Considérez les suites de chiffres suivantes : 1010, 102, 1084, ABC, 1AFD, 2AG, HE2
Lesquelles peuvent représenter un nombre en base 2, 8, ou 16 ?

Base 2 : Seulement 1010.
Base 8 : 1010, 102.
Base 16 : 1010, 102, 1084, ABC, 1AFD.
Quelle est la plus petite base pouvant représenter toutes ces suites ?

Toutes peuvent être représentées en base 18.

EXERCICE 3 :
A = (7365)8, B = (77,21)8, C = (F1E)16, D = (2DB,5)16.

C = (7436)8,

EXERCICE 4 :
Effectuer les opérations binaires :
1- 110011101 + 10110111 = 1001010100;
111001,1 + 11010,101 = 1010100.001;
2- 1010001,01 − 101110,1 = 100010.11;
1100101001 − 110110110 = 101110011;
3- 110110 × 101 = 100001110;
10111,011 × 101,1 = 10000000.1001;
4- 1111001111 ÷ 1101 = 1001011;
1110111111,11 ÷ 1011 = 1010111.01.
